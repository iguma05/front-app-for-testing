## Описание

Это React-приложение для практики мануального и автоматизированного тестирования.

## Структура проекта

```
├── src/
│   ├── components/          # React компоненты
│   ├── pages/              # Страницы приложения
│   ├── store/              # Redux store и API
│   ├── types/              # TypeScript типы
│   ├── hooks/              # Кастомные хуки
│   └── constants/          # Константы приложения
├── server/                 # BFF
├── public/                 # Статические файлы
└── build/                  # Собранное приложение
```

## Технологический стек

### Frontend

- **React 19** - библиотека для создания пользовательских интерфейсов
- **TypeScript** - типизированный JavaScript
- **Redux Toolkit** - управление состоянием приложения
- **RTK Query** - работа с API и кэширование данных
- **React Router** - маршрутизация в приложении
- **Material-UI** - компоненты пользовательского интерфейса
- **React Hook Form** - управление формами
- **Yup** - валидация данных
- **Axios** - HTTP клиент для API запросов

### Backend

- **Vercel Serverless Functions** - бессерверные функции для API
- **Node.js** - серверная среда выполнения

### Инструменты разработки

- **Create React App** - инструментарий для создания React приложений
- **ESLint** - линтер для JavaScript/TypeScript
- **Prettier** - форматирование кода

### Запуск в режиме разработки

```bash
# Запуск приложения на localhost:3000
npm start
```

### Сборка для продакшена

```bash
# Создание оптимизированной сборки
npm run build
```

Собранные файлы будут находиться в папке `build/`.

### Запуск тестов

```bash
# Запуск тестов
npm test
```

## Разработка

### Структура компонентов

- Все компоненты находятся в папке `src/components/`
- Стили компонентов вынесены в отдельные CSS файлы

### Управление состоянием

- Используется Redux Toolkit для глобального состояния
- RTK Query для работы с API
- Redux Persist для сохранения состояния между сессиями
